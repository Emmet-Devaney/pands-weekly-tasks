#Account Number Masking

## Description
This Python script hides the first six digits of a 10-digit account number provided by the user and displays the last four digits.

## How to Use
1. Run the script using Python.
2. Enter your 10-digit account number.
3. The script will hide the first six digits of the account number and display the last four digits.


##Notes
- Ensure that you enter a 10-digit account number.
- This script is intended to protect sensitive information by hiding the majority of the account number.
