# Balance Calculator

## Description
This script calculates the total balance in euro, based on two amounts provided by the user in cents.

## How to Use
1. Run the script using Python.
2. Enter the first amount you would like to add (in cents) when prompted.
3. Enter the second amount you would like to add ( in cents ) when prompted.
4. The script will then calculate the total balance in euro and print it.

## Notes
- The amounts entered should be in cents (e.g., 2500 for €25.00).
- The script divides the sum of the amounts by 100 to convert from cents to euro.
- The total balance is displayed with the euro symbol (€).
